Installation

Installing PyroCMS

  1. Download PyroCMS and extract the files.
  2. Upload the files to your web-server.

  3. Apply the following file permissions (Mac/Linux/BSD/etc only):

    cd /path/to/pyrocms/webroot/

    chmod 777 system/codeigniter/cache
    chmod 777 system/codeigniter/logs
    chmod 777 system/pyrocms/cache
    chmod 777 system/pyrocms/cache/dwoo
    chmod 777 system/pyrocms/cache/dwoo/compiled
    chmod 777 system/pyrocms/cache/simplepie
    chmod 777 system/pyrocms/config
    chmod 777 uploads

    chmod 666 application/config/config.php

  4. Navigate to your installation, E.g: http://example.com/ and you will automatically be redirected to the installer.

    Note: If this fails for any reason, go to http://example.com/installer/

  5. Follow all instructions on the screen, the when you are finished click Control Panel link.
  6. When you have confirmed the install worked, delete the /installer/ directory.
